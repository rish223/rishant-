An intelligent, AI-based study planner web application that helps students prepare effectively for their exams.

## Features

- **Smart Planning**: Generates personalized daily study schedules based on time available
- **Adaptive Schedules**: Different strategies for short-term (3 days), medium-term (1 week), and long-term (2+ weeks) preparation
- **Balanced Approach**: Includes study sessions, breaks, revision days, and rest days
- **No Sign-up Required**: Completely free, no login or payment needed
- **Beautiful UI**: Clean, student-friendly design with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Enter your exam name (e.g., "Mathematics Final Exam")
3. Select your exam date
4. Click "Generate Study Plan"
5. Review your personalized study schedule!

## Study Plan Features

- **Study Days**: Detailed daily schedules with timed sessions and breaks
- **Rest Days**: Scheduled recovery days (every 7 days) to prevent burnout
- **Revision Days**: Dedicated review sessions before the exam
- **Time Management**: Each task includes specific time allocations

## Technologies Used

- HTML5
- CSS3 (with modern gradients and animations)
- Vanilla JavaScript (no dependencies required)

## Demo

Simply open the `index.html` file in any modern web browser - no server or installation required!
